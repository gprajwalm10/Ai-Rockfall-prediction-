# AI-Based Rockfall Prediction & Alert System

A machine learning system that analyzes environmental and sensor data to predict rockfall risk and trigger early-warning alerts — reducing reliance on manual monitoring in hazardous zones.

## What It Does

- **Risk Prediction** — analyzes environmental parameters and sensor readings to generate rockfall probability scores
- **Early Warning Alerts** — automated alert system notifies when risk crosses defined thresholds
- **Real-Time Dashboard** — web-based interface for monitoring, visualization, and reporting of prediction outputs
- **Data Pipeline** — end-to-end processing from raw sensor data to actionable predictions

## Tech Stack

| Layer | Technology |

## Getting Started

```bash
# Clone the repo
git clone https://github.com/gprajwalm10/Ai-Rockfall-prediction-.git
cd Ai-Rockfall-prediction-/MineGuardAI-main

# Install dependencies
pip install -r requirements.txt

# Run the prediction system
python main.py
```

## How It Works

1. **Data Ingestion** — environmental and sensor data is fed into the pipeline
2. **Feature Engineering** — raw data is processed and transformed into model-ready features
3. **Prediction** — ML model generates rockfall risk scores
4. **Alerting** — automated alerts triggered when risk exceeds thresholds
5. **Dashboard** — results visualized in real-time on the web interface
